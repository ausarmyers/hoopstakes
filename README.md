# Hoopstakes

Quick setup and run instructions for Hoopstakes.

Prerequisites
- Node.js 16+ (or compatible LTS)
- npm or yarn
- Xcode / Android Studio for native builds (optional)

Clone

git clone https://github.com/ausarmyers/hoopstakes.git
cd hoopstakes

Install

npm install
# or
yarn install

Environment
- Copy any required env files (example: `.env`) and set secrets.

Local development

npm start
# or
yarn start

Android (device/emulator)

npm run android

iOS (device/simulator)

npm run ios

Notes
- This repo uses Expo/React Native and native android/ios folders.
- See `app/` for the React Native app source and `functions/` for Cloud Functions.
